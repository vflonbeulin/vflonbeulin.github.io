Soclia

The first AI platform for legal analysis

Gpt-4o NLP Django Azure
2024

Soclia is an AI SaaS solution for legal analysis. It was created in partnership with a notary.

My job was mainly on AI, backend and cloud services.

AI

The main goal was to use AI to read and understand legal documents concerning social landlords. I used NLP and computer vision, But the first results were not convincing, so i tried using an LLM (Gpt-4o because OpenAI is an azure partner) and fine tuned with functions calling.

Backend

I used PostgreSQL for the data and Django Rest Framework for the backend. API is RESTful, We can connect it to any frontend. I worked with a React developer and I gave some ideas for the UI. I've secured endpoints with JWTokens.

I suggested moving to Azure for infrastructure, to bring my certifications and Azure cloud architecture experiences to this project. I've created several services, including :

  • Azure App Service
  • Azure Storage
  • Azure Database PostgreSQL
  • Azure Container Registry
  • Azure AI (and Custom Vision)
  • Azure Cache Redis
    • etc... and more

I used Google Maps API to geolocate the addresses building of landlords. We used OpenStreetMap to display this information.

The app is containerized with Docker and push to a private registry.

Success

Our team has been selected by Éric Larchevêque in B3 village by CA, Vierzon, to continue the project.
